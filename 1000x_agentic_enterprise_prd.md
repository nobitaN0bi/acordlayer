### **3.4 Multi-Agent Debate Protocol**

#### **Consensus Mechanism**

```python
class DebateEngine:
    def resolve_disagreement(agents, problem, context):
        """
        When agents disagree on action, run structured debate
        """
        
        # Round 1: Initial Positions
        positions = {}
        for agent in agents:
            position = agent.propose_action(problem, context)
            positions[agent.id] = {
                "action": position.action,
                "confidence": position.confidence,
                "reasoning": position.reasoning
            }
        
        # Round 2: Challenge Phase
        challenges = {}
        for agent in agents:
            challenge = agent.challenge_others(positions)
            challenges[agent.id] = challenge
        
        # Round 3: Rebuttal & Confidence Adjustment
        updated_positions = {}
        for agent in agents:
            updated = agent.respond_to_challenges(challenges, positions)
            updated_positions[agent.id] = updated
        
        # Round 4: Consensus Decision
        decision = compute_consensus(updated_positions)
        
        return {
            "action": decision.action,
            "confidence": decision.confidence,
            "rationale": decision.rationale,
            "dissenting_opinions": decision.minority_views
        }
```

```
## DESIGN TOKENS (Colors, Typography, Spacing)

### **Color Palette**

```
Primary Colors:
- Teal (#2D8A88) - Action, CTAs, active states
- White (#FFFFFF) - Primary background
- Off-white (#F8F9FA) - Card backgrounds
- Gray (#666666) - Secondary text
- Near-black (#1A1A1A) - Primary text

Status Colors:
- Success (#2ECC71) - Positive outcomes, online status
- Warning (#F39C12) - Alerts, warnings
- Error (#E74C3C) - Errors, critical issues
- Info (#3498DB) - Informational messages
- Neutral (#95A5A6) - Disabled, inactive

Sentiment Colors:
- Positive performance: #2ECC71 (green)
- Neutral: #F39C12 (amber)
- Negative: #E74C3C (red)
```

### **Typography Scale**

```
Display: 32px bold (Inter), line-height 1.2
Heading 1: 28px semi-bold, line-height 1.2
Heading 2: 24px semi-bold, line-height 1.3
Heading 3: 20px semi-bold, line-height 1.4
Subheading: 16px semi-bold, line-height 1.4
Body: 14px regular, line-height 1.5
Caption: 12px regular, line-height 1.6
Code: 12px monospace (Monaco), line-height 1.5
```

### **Spacing Scale**

```
4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 48px, 64px
(Multiples of 4 for consistency)
```

### **Component Library**

```
Buttons:
- Primary (Teal, full width for CTAs)
- Secondary (White background, teal text)
- Tertiary (Text only, no background)
- Danger (Red for destructive actions)
- Sizes: Small (32px height), Medium (40px), Large (48px)

Cards:
- White background, subtle shadow (1px 2px 4px rgba(0,0,0,0.1))
- Padding: 24px
- Border radius: 8px
- Hover: +2px shadow

Forms:
- Input height: 40px
- Padding: 12px 16px
- Border: 1px #E0E0E0
- Focus: 2px teal border, shadow
- Label: 12px gray above input

Tables:
- Row height: 48px
- Header: Bold, teal background
- Striped rows: alternating white/off-white
- Hover row: +2px shadow
